# HR Analytics — Analyse de l'Attrition des Employés

## Description
Dans ce projet j'ai analysé les données RH d'une entreprise pour comprendre 
pourquoi les employés quittent l'entreprise et prédire les futurs départs.
J'ai utilisé le dataset IBM HR Employee Attrition disponible sur Kaggle.

## Outils utilisés
- Excel : calcul des KPIs, tableau croisé et graphiques
- SQL (SQLite) : 7 requêtes d'analyse
- Python (Google Colab) : analyse des données et Machine Learning
- Power BI : dashboard interactif

## Ce que j'ai découvert
- 16% des employés quittent l'entreprise
- Les employés qui partent sont plus jeunes et moins bien payés
- Le département Sales est le plus touché
- Les heures supplémentaires triplent le risque de départ

## Modèles Machine Learning utilisés
- Régression Logistique
- Random Forest
- XGBoost (meilleur modèle)

## Dataset
IBM HR Employee Attrition — Kaggle
